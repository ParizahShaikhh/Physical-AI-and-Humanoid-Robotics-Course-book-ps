# Quickstart: Module 2 - The Digital Twin (Gazebo & Unity)

## Overview
Module 2 focuses on digital twins for humanoid robot simulation using Gazebo and Unity. This module teaches students how to use simulation environments for safe, repeatable testing of physics, sensors, and interactions in Physical AI development.

## Prerequisites
- Completion of Module 1 (ROS 2 fundamentals)
- Basic understanding of robotics concepts
- Familiarity with simulation concepts is helpful but not required

## Getting Started

### 3. Key Learning Objectives
By the end of this module, students will be able to:
- Explain the role of digital twins in robotics
- Simulate physics, gravity, and collisions in Gazebo
- Build interactive environments in Unity
- Understand and configure simulated sensors (LiDAR, depth cameras, IMUs)
