# Module 2: The Digital Twin (Gazebo & Unity)

## Overview

Welcome to Module 2 of the Physical AI and Humanoid Robotics Course. This module focuses on digital twins for humanoid robot simulation using Gazebo and Unity. You'll learn how to use simulation environments for safe, repeatable testing of physics, sensors, and interactions in Physical AI development.

## Learning Objectives

By the end of this module, you will be able to:
- Explain the role of digital twins in robotics
- Simulate physics, gravity, and collisions in Gazebo
- Build interactive environments in Unity
- Understand and configure simulated sensors (LiDAR, depth cameras, IMUs)

## Prerequisites

- Completion of Module 1 (ROS 2 fundamentals)
- Basic understanding of robotics concepts
- Familiarity with simulation concepts is helpful but not required
